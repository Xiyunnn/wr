# GENOM SCRIPT VPS INSTALL


## 1. INSTALL

```
 apt update && apt upgrade -y && wget -q https://github.com/Xiyunnn/wr/raw/refs/heads/main/install && chmod +x install  && ./install
```
## 2. UPDATE 

```
cd root
rm update.sh
wget https://github.com/Xiyunnn/wr/raw/refs/heads/main/menu/update.sh && chmod +x update.sh && ./update.sh
```
